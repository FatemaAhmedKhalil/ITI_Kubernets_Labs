## 📖 Overview

This repository contains hands-on labs and exercises for a **Kubernetes (K8s) course**.

Each lab is designed to build practical experience with core Kubernetes concepts, including cluster setup, deployments, services, networking, and troubleshooting.

The goal is to strengthen understanding of Kubernetes through real-world scenarios and step-by-step practice.

---

## 🚀 Topics Covered

* Kubernetes cluster setup (k3s / kubeadm)
* Pods and Deployments
* Services and Networking
* ConfigMaps and Secrets
* Volumes and Persistent Storage
* Debugging and Troubleshooting
* Basic DevOps workflows

---

## 🛠️ Tools Used

* Kubernetes (k8s)
* k3s
* kubectl
* Linux (Ubuntu)
